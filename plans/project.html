Project Description

Music disc jockeys and other remix artists specialize in the layering of different songs and tracks to create popular remixes and to build 'hype' through music.

While there are many songs that mix well together, many do not. Here lies the creative process. Artists take many technical factors into consideration when mixing several songs.

These include:
  • Tempo
  • Key
  • Chord Progression
  • Genre

To aid artists in their creation process, we are creating a tool that allows users to find songs based on these properties.

Usefulness

This application will enable artists to find music in a given key, tempo, and genre to mix well with an already existing project or idea. Through this we hope to facilitate the creative process.

There are several existing applications similar to ours:
  1. Tunebat
    • Allows users to search for songs to find bpm and key.
    • Users can find recommended songs based on an inputted song.
  2. Spotify
    • Allows users to find recommended music based on current playlists, liked songs, or previously listened to music.
    • Has an advanced API for searching music and facilitating the above bullet.
    • Lacks features to find music based on key and tempo.
  3. Discz
    • Allows users to upload a current playlist and display song recommendedations based on playlist
    • Users can swipe on recommended songs wether they like the song or not.

Our goal is to improve on these applications by generating a list of similar recommended songs that mix well (similar tempo, key, chord progression) with an inputted song. This list will be sorted based on latest popularity, and have options to listen to songs mixed automatically.

Realness - Data

We will gather data from one or more of the following sources:

  • Scraping data from Tunebat or Spotify using an API.
  • The million song dataset found here (published 2011).

Functionality

  • Basic Functions:
    1. Users can add songs and metadata information that are not currently in the database.
    2. Users can search by key, tempo, release year, and other properties to find recommended songs.
  • Advanced Functions
    1. Users can sort searched results by current song popularity. Popularity will be obtained from sites like spotify and apple music top charts.
    2. Users can auto generate playlists where songs auto fade in and fade out with each other.
