  4. HTML / CSS / Javascript / Bootstrap - frontend styling

Data

To collect our data we will pull from several sources

  1. Song metadata will be scraped from the spotify api.  This includes Key, Tempo, danceability, loudness, and other information
  2. Song titles, artists, albums, and other information will be loaded from the MusicBrainz open source music database.  This contains over 27.5 million entries and characterizes relationships between artist, albums, releases and other data

Our overall data collection process will be as follows:

  • Load songs, artists, and albums from the downloaded database
  • Scrape spotify api for key, tempo, and other metadata operations
  • Store concatenated data to our database
